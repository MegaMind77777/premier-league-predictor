
# Premier League Match Predictor

## Overview
Analysis and prediction of Premier League match results using 20 years of data (2001-2022).
Built a machine learning model to predict whether a match results in a Home Win, Away Win, or Draw.

## Question
Can we predict Premier League match results using pre-match statistics?

## Key Findings
1. Home advantage is real — home teams win 46.4% of matches
2. Form is a strong predictor — winning teams average 6.90 form points vs 5.43 for losing teams (27% difference)
3. Points difference matters — stronger teams win more often but upsets happen
4. Man United and Chelsea are the highest home scorers over the 20 year period

## Model Performance
- Algorithm: Random Forest Classifier
- Accuracy: 47.6% (vs 33.3% random baseline — 43% better than chance)
- Draws were hardest to predict — consistent with football analytics research
- Home wins predicted most accurately
